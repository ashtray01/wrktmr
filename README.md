# worktimer
routine manager in terminal
<br />простой менеджер задач в терминале
<br />
<br />Вводи задачи.
<br />Для завершённых задач начни с **!**
<br />Команды:
<br />
<br />:home :h — завершить день
<br />:s :save — сохранить в Excel
<br />:q :quit :exit — выйти
